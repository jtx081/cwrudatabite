What is CWRU Databite?

The CWRU Databite is a website that allows users to upload data for preprocessing and cleaning. Oftentimes data cleaning, data preprocessing, and visualization are time consuming, yet essential in data analytics. CWRU Databite has come up with data processing and visualization tools useful for users even without technical background and unfamiliar with the process. After uploading and processing the data, the user will be able to visualize that data on the website and export the cleaned data.

Usage

Login Page

To login to your created account, enter your email as the username and then your password.

Sign Up Page

In the sign up page, you can choose which type of account to sign up for. You can create an individual account by entering an email address and password. If you are registering for an account to be used under a company, you can also fill in the optional company ID box to join the company.

If you wish to register a new company, you can select the business account option in the login page. This will bring up a sign up form which includes an extra box to enter your company name.

Data Upload

This option is available on the main page as the button “Upload New Item”. In order to upload a file you select a file using the file browsing window.

Acceptable file types include:

CSV, TSV, txt, dat, sql, xml

The “View Datasets” button needs to be clicked in order to display the existing datasets that are stored.

Visualizing data

After logging into your account, you will have the option to upload datasets for cleaning. Click on “View Datasets” to see previously uploaded data. For each row, you can have the option to save the data to a local disk, remove duplicate data, normalize the data and standardize data. You may also choose to visualize data using the graph button.

Data Download

After the “View Datasets” button is clicked, under Actions, you can click the “Download Dataset” button to download the dataset as the file type it is saved as in its current state.

Account page

For individual users they will see the My Account tab with the options to change their password or delete their account. To change your password, you enter your new password twice and then click the “Change Password” button. The “Delete My Account” button will prompt you for confirmation for deleting your account. Upon confirmation, your account will be deleted and you will be redirected to the default login/signup page.

The Manageable Users page is only visible to users who are company admins. In order to view users in the company, the user will have to click on the “View Users” button. They have the option to remove accounts who are not company admins and under their company. They can also upgrade existing accounts to company admin accounts for the company.
